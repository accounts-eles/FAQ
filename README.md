🔍 Interactive FAQ Accordion

A sleek, modern FAQ component designed for clarity and ease of use. This project features a responsive accordion-style interface that allows users to toggle between questions and answers seamlessly, providing a professional way to handle frequently asked questions on any website.

✨ Project Overview

The FAQ component is built with a focus on user experience and accessibility. It utilizes a "clean-reveal" interaction pattern, ensuring that large amounts of information remain digestible by hiding details until requested.

Key Features

Accordion Interaction: Smooth, vertical expansion and contraction of content areas using CSS max-height transitions for a fluid motion feel.

Visual State Cues: Integrated dynamic plus/minus indicators that toggle based on the active state of each question.

Asset Placeholders: Built-in slots for supporting imagery and icons, allowing for rich-media answers that go beyond simple text.

Branded Styling: Uses a sophisticated professional palette:

Midnight Blue (#1f2a52): Primary headers and grounding.

Accent Cyan (#00bec7): Borders and interactive highlights.

Light Aqua (#d2f0f0): Soft background for answer panels.

Responsive Typography: Powered by the Inter font family, ensuring high readability across all screen sizes.

🛠️ Technical Implementation

Vanilla JavaScript: Lightweight logic handles the toggling of active states and ensures only one section remains open at a time for a focused user experience.

Tailwind CSS: Utilizes utility-first styling for rapid layout development, supplemented by custom CSS for complex transitions.

Modern Web Standards: Implements DOMContentLoaded listeners and semantic <button> elements for accessibility.
